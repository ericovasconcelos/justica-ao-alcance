## ⚡ PRÓXIMOS 3 PASSOS (URGENTE)

### PASSO 1: Publicar o Site (HOJE - 29/09)
**Quem**: Qualquer pessoa do grupo com computador

1. Abra o arquivo `GUIA_PUBLICACAO.md`
2. Siga as instruções da seção "Passo 2: Criar um Novo Repositório"
3. Faça upload dos 3 arquivos: `index.html`, `styles.css`, `script.js`
4. Ative o GitHub Pages
5. Copie a URL que aparecer (exemplo: https://seu-usuario.github.io/justica-ao-alcance)
6. Compartilhe a URL no grupo do WhatsApp

**Tempo estimado**: 15-20 minutos

---

### PASSO 2: Gerar QR Code (AMANHÃ - 30/09)
**Quem**: Mesma pessoa do Passo 1 ou outra

1. Acesse: https://www.qr-code-generator.com/
2. Cole a URL do site que você publicou
3. Clique em "Criar QR Code"
4. Customize:
   - Cor: #8B4789 (roxo)
   - Tamanho: Grande (para impressão)
5. Baixe o QR Code em alta resolução
6. Teste com seu celular (escaneie para ver se abre o site)
7. Envie o arquivo para a Bárbara

**Tempo estimado**: 5-10 minutos

---

### PASSO 3: Criar Panfleto (01-02/10)
**Quem**: Bárbara Radtke

1. Abra o arquivo `DESIGN_PANFLETO.md` para ver as sugestões
2. Entre no Canva.com (gratuito)
3. Crie um design de panfleto A5 ou A6
4. Inclua:
   - Título: "Justiça ao Alcance"
   - Números: 190, 180, 197
   - QR Code (que você recebeu no Passo 2)
   - Texto: "Escaneie para mais informações"
   - Logo da UDF
5. Exporte em PDF (alta qualidade)
6. Compartilhe com o grupo para aprovação

**Tempo estimado**: 1-2 horas

## 🆘 PRECISA DE AJUDA?

### Se o site não funcionar:
1. Verifique se os arquivos estão com os nomes exatos: `index.html`, `styles.css`, `script.js`
2. Aguarde 10 minutos após ativar o GitHub Pages
3. Limpe o cache do navegador (Ctrl + Shift + Delete)
4. Tente acessar em modo anônimo

### Se o QR Code não funcionar:
1. Verifique se você colou a URL correta
2. Teste o QR Code escaneando com seu celular
3. Se não funcionar, gere um novo

### Se tiver dúvidas sobre design:
1. Consulte o arquivo `DESIGN_PANFLETO.md`
2. Peça feedback no grupo antes de imprimir
3. Sempre faça uma impressão teste primeiro
