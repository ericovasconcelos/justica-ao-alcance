# 🎨 SUGESTÕES DE DESIGN PARA O PANFLETO

## Formato Sugerido

**Tamanho**: A5 (14,8 x 21 cm) ou A6 (10,5 x 14,8 cm)
- Fácil de distribuir
- Cabe em bolsas e bolsos
- Custo de impressão reduzido

## Elementos Visuais

### Paleta de Cores
- **Cor Principal**: #8B4789 (Roxo)
- **Cor Secundária**: #C084BC (Lilás)
- **Cor de Destaque**: #E91E63 (Rosa)
- **Texto**: #2C2C2C (Cinza Escuro)
- **Fundo**: #FFFFFF (Branco)

### Tipografia
- **Título**: Fonte bold, tamanho grande (24-36pt)
- **Subtítulos**: Fonte semibold (14-18pt)
- **Corpo**: Fonte regular (10-12pt)
- **Sugestões de fontes**: Montserrat, Poppins, Open Sans, Roboto

## Elementos Gráficos Sugeridos

### Ícones
- 🌹 Rosa (símbolo de força e resistência)
- 🚨 Alerta/Sirene (para emergências)
- 💜 Coração (para apoio emocional)
- 📞 Telefone (para contatos)
- ⚖️ Balança (justiça)
- 🛡️ Escudo (proteção)

### Imagens
**EVITE:**
- Imagens de violência explícita
- Fotos de mulheres machucadas
- Imagens que causem desconforto

**PREFIRA:**
- Silhuetas femininas em postura de força
- Mãos estendidas (apoio)
- Círculo de mulheres unidas
- Símbolos abstratos de união e força
- Gradientes suaves com as cores do projeto

## Dicas de Design

### Hierarquia Visual
1. **Mais importante**: Números de emergência (maior tamanho)
2. **Importante**: QR Code e título
3. **Secundário**: Informações complementares

### Espaçamento
- Não sobrecarregue o panfleto
- Deixe "respiro" entre os elementos
- Use margens adequadas (mínimo 5mm)

### Contraste
- Texto escuro (#2C2C2C) sobre fundo claro
- Destaque os números de emergência com cor de fundo
- Use caixas coloridas para separar seções

### Legibilidade
- Tamanho mínimo de fonte: 10pt
- Fonte sem serifa para melhor leitura
- Evite itálico em textos longos
- Use negrito para destacar informações importantes

## Checklist Antes de Imprimir

### Verificações Técnicas
- [ ] Resolução mínima: 300 DPI
- [ ] Modo de cor: CMYK (para impressão)
- [ ] Sangria: 3mm em todos os lados
- [ ] Fontes convertidas em curvas/vetores
- [ ] QR Code testado e funcionando
- [ ] Margens de segurança respeitadas

### Verificações de Conteúdo
- [ ] Todos os números de telefone corretos
- [ ] URL do site correta no QR Code
- [ ] Sem erros ortográficos
- [ ] Informações atualizadas
- [ ] Logo da UDF incluído
- [ ] Identificação do projeto clara

### Teste de Impressão
- [ ] Imprimir 1 cópia teste em preto e branco
- [ ] Imprimir 1 cópia teste colorida
- [ ] Verificar se o QR Code está legível
- [ ] Confirmar que os textos estão nítidos
- [ ] Validar tamanho final do panfleto
